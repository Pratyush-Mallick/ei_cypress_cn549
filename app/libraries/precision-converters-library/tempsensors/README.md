# tempsensors

[Analog Devices Inc.](http://www.analog.com/en/index.html) Temperature sensors library

## About
This library contains the functionality to interface with various temperature sensors as listed below:
- NTC 44031 (10K)
- PTC ky81
- Thermistors
- Thermocouples
- RTDs
